Instructions to run the file:

################## PyCharm IDE preferred #################################

-> Extract the "proj3p2_group20_python.zip" to any desired folder.

-> 1. Using Command Prompt:

->-> Navigate to the "codes" folder inside "proj3p2_group20_python" folder, right click and select open terminal or command prompt here.
	*** use 'python3' instead of 'python' in the terminal if using Linux based OS ***
	Execute the files by using the following commands in command prompt or terminal:
	--python .\Astar_rigid.py   
	-- Enter the Start Point parameters in single line terminal input as (50, 30 ,60)
	-- Enter the Goal Point parameters in single line terminal input as (150, 150, no input or 0)
	-- Enter the Robot Radius as 1
	-- Enter the Robot Clearance as 1
	-- Enter the Robot Step Size as 1 as well.
	

-> 2. Using PyCharm IDE

	Open all the files in the IDE and click Run and select the "Astar_rigid.py" to execute.
	-- Enter the Start Point parameters in single line terminal input as (50, 30 ,60)
	-- Enter the Goal Point parameters in single line terminal input as (150, 150, no input or 0)
	-- Enter the Robot Radius as 1
	-- Enter the Robot Clearance as 1
	-- Enter the Robot Step Size as 1 as well.
